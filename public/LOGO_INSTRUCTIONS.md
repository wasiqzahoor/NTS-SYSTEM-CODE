# Custom Logo Setup

Place your custom logo file as:
  /public/logo.png

Requirements:
- File name: logo.png
- Recommended size: 512x512px or 256x256px
- Format: PNG with transparent background preferred
- The logo will appear in the sidebar at 40x40px

Steps:
1. Copy your logo.png into this /public/ folder
2. Restart the dev server
3. The sidebar will automatically show your logo

If no logo.png is found, a default gradient icon is shown as fallback.
